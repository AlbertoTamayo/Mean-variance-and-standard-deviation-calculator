# Mean-Variance-Standard Deviation Calculator

The mean_var_std.py file contains the main code that uses numpy to reshape an input list containing nine numbers into a 3x3 numpy array and computes mean, variance, standard deviation, etc. The test_module.py file contains various tests for the mean_var_std.py function that must be fulfilled in order to verify correct functionality of mena_var_std.py The main.py file runs the mean_var_std.py and the test_module.py automatically
